Boocle
======

輪読管理サービス

======

React + Backbone.js + Parse を利用しています

React
http://facebook.github.io/react/

Backbone
http://backbonejs.org/

Parse
https://parse.com/

======

このままWEBサーバに乗せるだけで動作します
データ及び一部のクラウドコードに関してはデフォルトで本番環境に接続されます
(Parseにアカウントを作成すれば自分用の環境を作成することも可能です)
